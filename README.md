# UNSENT — The Unsaid Collection

A static personal-message website built with HTML, CSS, and JavaScript.

## Run locally

Open `index.html` in a browser.

## Publish with GitHub Pages

1. Upload the **contents of this folder** to the root of a GitHub repository.
2. In the repository, open **Settings → Pages**.
3. Select **Deploy from a branch**, choose the main branch and `/ (root)`, then save.

## Branding assets

The official logo files are stored in `assets/branding/`:

- `unsent-logo-full.png` — original generated brand presentation
- `unsent-logo-lockup.png` — cropped full logo and tagline
- `unsent-mark.png` — emblem used inside the website
- `favicon.ico` and PNG favicon sizes — browser and device icons

## Important limitation

This is a static frontend project. Invitation codes and one-time-read status are not protected by a backend database. The read status is stored in the visitor's browser.
